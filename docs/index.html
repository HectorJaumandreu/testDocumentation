recipeDetails

Function that retreives the information about the recipe. It takes the information from the proxy if it is existent and the proxy flag is not false. It takes the information form the API if it does not exist or the proxy flag is false. This function represents the Model in the MVC pattern. As is an asyncronous call it does not return values and triggers the controller and the view.

recipeDetails(recipeId: string)
Parameters
recipeId (string) Id of the recipe
